Procedural grass from Ghost Of Tsushima

This project is built using a custom graphics engine using primarily Rust and WebGPU.

Some things i learned from this project was

  • Creatin a custom graphics engine
  • How the render pipeline works
  • Compute shaders
  • Procedurally generating vertices
  • Phong shading with custom specular blending with distance
  • Compiling to WebAssembly
